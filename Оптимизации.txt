ALIGN - выравнивание данных в памяти

align 4; размещать данные по адресам кратным 4

XOR меняет регист признаков
INC быстрее чем ADD


Вместо 
cmp eax, 0
jz is_zero 
можно использовать
or eax, eax
jz is_zero


Для проверки числа на отрицательность можно использовать TEST
TEST eax, eax
js is_negative

Бит SF == 1, если знак отрицательный
